# 交换机
想着既然有路由器那交换机也应该有吧，交换机是局域网内的设备，主要功能是连接局域网内的设备，并且根据MAC地址进行数据转发。交换机工作在OSI模型的第二层（数据链路层），它通过学习和维护一个MAC地址表来决定将数据包转发到哪个端口。交换机的主要功能包括：
1. **数据转发**：交换机根据MAC地址表将数据包转发到正确的端口。
2. **广播域隔离**：交换机可以通过VLAN（虚拟局域网）技术将一个物理网络划分成多个逻辑网络，从而隔离广播域。
3. **全双工通信**：交换机支持全双工通信，可以同时发送和接收数据，提高网络效率。
4. **MAC地址学习**：交换机通过监听网络流量来学习设备的MAC地址，并将其记录在MAC地址表中，以便进行正确的数据转发。
交换机的类型主要有两种：非管理型交换机和管理型交换机。非管理型交换机简单易用，适合家庭和小型办公室使用；管理型交换机则提供更多的功能和配置选项，适合企业级网络环境。
